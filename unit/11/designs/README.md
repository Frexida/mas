## デザイン原則

### 1. ユーザー中心設計
- ユーザーのゴールを明確に
- 認知負荷を最小化
- 直感的な操作性

### 2. 一貫性
- 統一されたデザイン言語
- 予測可能なインタラクション
- 標準的なパターンの使用

### 3. アクセシビリティ
- WCAG 2.1 AA準拠
- キーボードナビゲーション
- スクリーンリーダー対応

### 4. レスポンシブ
- モバイルファースト
- 流動的なレイアウト
- タッチ操作最適化

### 5. パフォーマンス
- 軽量なアセット
- 遅延ロード
- スムーズなアニメーション
